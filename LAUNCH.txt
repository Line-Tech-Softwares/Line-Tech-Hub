⚙️ FIREBASE SETUP (REQUIRED FIRST)
─────────────────────────────────────────────────────────────────────────────

STEP 1: Create Firebase Project
  Go to: https://console.firebase.google.com
  Create new project: "Line Tech Hub"
  
STEP 2: Enable Services
  - Authentication (Email + Google)
  - Firestore Database (Test Mode)
  - Cloud Storage (Test Mode)
  
STEP 3: Whitelist Localhost
  Authentication → Settings → Authorized Domains
  Add: localhost:5173
  
STEP 4: Copy Credentials
  Project Settings → Web App → Copy config
  
STEP 5: Create .env.local
  Edit: c:\LTS PROJECTS\Line Tech Hub\.env.local
  Paste your Firebase credentials:
  
  VITE_FIREBASE_API_KEY=AIzaSy...
  VITE_FIREBASE_AUTH_DOMAIN=...
  VITE_FIREBASE_PROJECT_ID=...
  VITE_FIREBASE_STORAGE_BUCKET=...
  VITE_FIREBASE_MESSAGING_SENDER_ID=...
  VITE_FIREBASE_APP_ID=...

See QUICKSTART.md for complete instructions.

❓ TROUBLESHOOTING QUICK FIXES
─────────────────────────────────────────────────────────────────────────────

"Firebase initialization failed"
  → Check .env.local exists with all 7 values
  
"Cannot connect to Firebase"
  → Verify credentials match Firebase Console exactly
  
"Blank page after login"
  → Firestore in Test Mode? Check Rules tab in Firebase
  
"@linetechsoftwares.co.za only" error on dev login
  → CORRECT! Use client login for external emails
  
"Port 5173 already in use"
  → npm run dev -- --port 5174
  
"npm install fails"
  → npm install --legacy-peer-deps

Full troubleshooting: See SETUP.md → "Troubleshooting"
